# Mini-Bank - банковский бэкенд с промышленными тестами

Учебный проект для отработки навыков написания автотестов в условиях, близких к реальным. Бэкенд банковской системы на Spring Boot с полным циклом тестирования.

## Суть проекта
Не просто "еще один учебный проект", а площадка для отработки hard skills в автотестировании бэкенда.

## Технический стек
- **Java 17** + **Spring Boot 3.2.0** - современный стек
- **JUnit 5** + **AssertJ** + **Mockito** - стандарт для enterprise
- **Gradle** (Kotlin DSL) - промышленная сборка
- **Spring Boot Test** - интеграционное тестирование

## Что реализовано

### Бизнес-логика
- Создание и управление банковскими счетами
- Переводы между счетами с валидацией
- История транзакций
- Обработка исключений с детальными JSON-ответами

### Тестовое покрытие
1. **Unit-тесты** моделей - проверка бизнес-правил (баланс, валидация)
2. **Unit-тесты** сервисов - изоляция логики с Mockito
3. **Интеграционные тесты** API - полный цикл HTTP запросов
4. **Тесты на граничные случаи** - негативные сценарии, ошибки
